# Related work:

## Selection in NNs:

- Almost all literature on selection in neural networks focuses only on sparsifying the neural network architecture itself, without much if any regard to input feature selection.

  - Different variations of dropout (pruning inputs and layer units during training) have long been used to prevent overfitting in deterministic NNs; Gal & Gahramani 2016 - Dropout as a Bayesian Approximation - show that dropout corresponds to an approximate variational distribution.

  - Louizos, Ullrich, Welling 2017 "Bayesian Compression" - which we draw most heavily from.  Their focus is on a grouped scale mixture prior, providing code for a Normal Jeffrey's prior.  Their method was motivated by the desire to compress neural network sizes, so focuses on pruning hidden units and using shorter bit-length representations of weights ("quantization") while preserving predictive accuracy.

  - Ghosh & Doshi-velez 2017/2019 model selection - prune _units_ with horseshoe, i.e. selection of number of neurons in each layer, not on input variables.
    - selection based on thresholding

  - Skaaret-Lund, Storvik, Hubin 2023 (related work from Hubin Storvik in 2019, 2023)- propose latent binary bayesian neural network (LBBNN) - use spike-and-slab with latent variables to model dependencies between weights; variational inference procedure enhanced with inverse autoregressive flow (type of normalizing flow)


- When variable selection is the actual focus, input sparsity / variable selection is still typically treated heuristically.  Of the very small handful of works that focus specifically on input feature selection:

    1. Liang et al 2018 Bayesian Neural Networks for Selection of Drug Sensitive Genes - 

      - 1 hidden layer, relatively small dimension (i.e. all results are from NNs with 3, 5, 7 units in hidden layer, chosen empirically after short MC run to minimize an information criterion - they use EBIC, but suggest BIC and AIC also work)  

      - selection performed over every single network connection --- authors assert that this offers some interpretability; Sure, but probably only in a single hidden layer network, or v. small / v. sparse architectures.  

      - population stochastic approximation monte carlo sampling - run short stochastic approximation MC chains in parallel by sharing information on a working parameter that penalizes over-visited regions and rewards under-visited regions to help system escape local extrema

    2. an ArXiv paper Overweg Popkes 2019 "Interpretable outcome prediction with sparse bayesian neural networks in intensive care" - similar approach to ours, but variable selection is still heuristic and based on defining thresholds after inspection of posterior distributions.

    3. Song and Li 2021 developed SurvNet, a non-Bayesian procedure employing synthetic input variables and backward elimination to control the FDR in deep NNs.

    4. Liu 2021: Variable Selection with Rigorous Uncertainty Quantification using Deep Bayesian Neural Networks: Posterior Concentration and Bernstein-von Mises Phenomenon - Uses standard iid Gaussian priors for model weights rather than priors specifically developed for sparsity, instead developing a BvM theorem for a measure of variable importance, the L2 norm of the graient function partial derivative signifying change in f(\vec{x}_i) over change in x_p, and performing variable selection by checking credible intervals for this variable importance measure.

- Last, some theory has been developed showing posterior consistency for model selection and asymptotic normality for NN's, though it is still relatively sparse.  Rockova & Polson 2018 show near-minimax-rate optimal posterior concentration over model architectures in Bayesian spike-and-slab deep NNs with RELU activations; Chrief-Abdellatif 2020 shows that the variational posterior learned through optimization of the ELBO shares these properties. Wang Rockova 2020 develop Bernstein von Mises theorems for linear and quadratic functionals in the same network types; Liu 2021 obtains further BvM theorems indicating nominal coverage of credible intervals from Bayesian NNs with simple Gaussian priors.


# Spike-and-slab & scale mixtures

- Scale mixtures as relaxed version of spike-slab (bernoulli computation explodes dimensionality and is thus costly).

- LUW 2017 use the Normal-Jeffrey's, but Hron et al 2009(?) show that the improper prior induces pathologies related to the VI approximation

## Horseshoe prior 
- posterior consistency, BMA-like properties for prediction, shrinkage factor $\kappa_j = (1 + \lambda_j^2 \tau^2)^{-1}$ and $1 - \kappa_j$ as "pseudo" PIPs (Carvalho 2009?)

- posterior consistency for architecture selection (Rockova Polson 2017?)


# Variational Inference

- VI used instead of costly MCMC; discussions from LUW 2017 and Ghosh & Doshi-Velez provide a quick overview

- Mean Field VI - 
  - fast, easy to compute if VI distributions are fully factorized / independent
  - some evidence that network depth can overcome the independence in the VI formulation (Farquhar Smith Gal 2020 Try Depth instead of weight correlations....)
  - limitations:
    - approximation is limited by variational distributions.  Log-Normal dist'n probably not the best; could possibly be improved by using normalizing flows as in Skaaret-lund & Storvik
    - Ghosh and Doshi-Velez 2019 address some of these by placing a MVN prior on weights, but not implemented here
